# Right After Landing Page

Career strategy and resources for lawyers who are done.

## About
Right After is a guide for lawyers looking to transition away from law practice. This repository contains the landing page.

## Contents
- `right-after-landing-v2.html` — Main landing page
- `index.html` — (Optional) Rename the landing page to this for GitHub Pages

## Setup & Deployment

### Local Development
Simply open `right-after-landing-v2.html` in your browser, or use a local server:
```bash
python -m http.server 8000
# or
npx http-server
```

### GitHub Pages
1. Rename `right-after-landing-v2.html` to `index.html`
2. In repo settings, enable GitHub Pages (source: main branch / root)
3. Your site will be live at `https://yourusername.github.io/right-after-landing/`
